## 使用N来管理nodejs的版本

### n

[n](https://www.npmjs.com/package/n) 是一个 Node.js 版本管理器，它非常简单。（n 目前不支持 Windows 环境）

### 安装

``` shell
$ npm install -g n
```

### 安装/使用某个版本

``` shell
$ n 0.8.14
$ n 0.8.17
$ n 0.9.6
```

### 查看已安装版本

``` shell
$ n
```

### 安装/使用最新版本

``` shell
$ n lastet
```

### 安装/使用稳定版本

``` shell
$ n stable
```

### 卸载某个版本

``` shell
$ n rm 0.9.6
```
